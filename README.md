This thesis presents the design and implementation of a fully decentralized card game relying on fair card shuffling and game flow without the presence of a trusted dealer. The idea was to create a system that mimics a real life card game outside the casino, where every player actively plays a part within the deck. The application is built combining multiple technologies to create a secure system to ensure that no participant can manipulate the deck, learn hidden cards, or alter the outcome of the game.
The game is an identical copy of the most notorious poker game in the world, Texas HoldEm Poker, using the exact hand rankings and rules as every other online or real life casino poker game. The way that this system is different compared to classical online casino games, is that all players have to participate in all stages, taking turns in shuffling the deck, dealing the cards, and revealing the other players hands. Each player contributes to the randomization of the deck through a cryptographic shuffle process, demonstrating the validity and correctness by the usage of Zero-Knowledge Proofs without revealing secret information.
